============================================
  API MeuFreelas - DEPLOY NA HOSTINGER
============================================

Para que https://meufreelas.com.br/api/setup.php
( e login/cadastro ) funcionem:

1. Copie TODA esta pasta "api" para a raiz do seu site
   no servidor, com o nome "api".

   Exemplo: se a raiz do site é public_html/
   → coloque os arquivos em: public_html/api/

   Resultado: public_html/api/setup.php, auth.php, etc.

2. Crie o arquivo api/.env no servidor com:
   DB_HOST=localhost
   DB_PORT=3306
   DB_NAME=u892594395_meufreelas
   DB_USER=u892594395_meufreelas27
   DB_PASS=sua_senha_mysql
   API_ORIGIN=https://meufreelas.com.br
   SMTP_HOST=smtp.hostinger.com
   SMTP_PORT=465
   SMTP_USER=noreply@meufreelas.com.br
   SMTP_PASS=senha_da_caixa_email
   SMTP_FROM=noreply@meufreelas.com.br
   SMTP_FROM_NAME=MeuFreelas

3. Acesse no navegador:
   https://meufreelas.com.br/api/setup.php
   Deve aparecer JSON com "ok": true.

Se você NÃO enviar a pasta api/, o link /api/setup.php
vai cair na página inicial do site (home). Envie a pasta.
