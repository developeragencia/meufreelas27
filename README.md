# MeuFreelas

Plataforma para conectar clientes e freelancers no Brasil.

- **Frontend:** pasta `app/` (React + Vite + TypeScript)
- **Deploy:** 100% Hostinger — ver **[DEPLOY-HOSTINGER.md](DEPLOY-HOSTINGER.md)** para build e publicação.

Não há configurações de Netlify ou outros provedores; apenas Hostinger.
